# Mistakes Log

When Claude makes a mistake, or when you make a decision you want to learn from, log it here. Claude reads this file at the start of every session so it doesn't repeat past errors.

**Rules:**
- Newest entries at the top
- Never delete entries — this is append-only
- Be specific about what went wrong and what to do differently

**Format:**
```
## YYYY-MM-DD — Short title

**What happened:** What went wrong, in plain language.
**What to do differently:** The concrete lesson.
**Tags:** #category
```
